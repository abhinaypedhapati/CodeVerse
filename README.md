## 🎯 How to Use

1. **Select Language** - Choose from 12+ programming languages
2. **Write Code** - Use the built-in editor with syntax highlighting
3. **Execute** - Click the Execute button for instant results
4. **Save Snippets** - Login to save your favorite code snippets
5. **Live Preview** - For HTML, see real-time visual output
